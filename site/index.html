Today, AI hands you a blob.

Ask an assistant to tweak a chart in a deck and it regenerates the whole thing as a flat image: no structure, no consistency, nothing you can touch. Sheep turns that artifact into parts.

Without Sheep

With Sheep

"Change the chart" → a patched field

Three calls, no redraw.

Components live in a registry with schemas, slots and stable ids. Your AI works them over standard MCP tools.

1

Discover

Find a component by intent. Each one ships a schema and a preview.

// MCP
discover("bar chart")
2

Instantiate

Drop it into the artifact with data. Style is theme-token only, so no font drift.

instantiate({
  type: "chart.bar",
  data: [...]
})
3

Patch

Edit one field by id. Transactional and reversible: an inverse op comes for free.

apply_patch({
  id: "chart-7",
  set: { "series[0].values": [...] }
})

Click an element. That's the scope.

Select any part of this slide. Sheep resolves it to a stable id and hands the AI exactly that node, nothing else.
